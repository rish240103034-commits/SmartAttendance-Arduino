# Smart RFID Attendance System using Arduino

A menu-driven RFID attendance system controlled completely using an IR remote.

Unlike traditional attendance systems, this project provides a TV-like user interface with LCD navigation and remote-controlled operation.

---

## Features

- RFID Based Attendance
- IR Remote Controlled Interface
- Power ON/OFF System
- Menu Navigation
- Attendance Session Control
- Student Records
- Attendance Report
- Live Attendance Count
- Reset Attendance
- Unknown Card Detection
- Duplicate Scan Prevention
- LCD User Interface
- Active Buzzer Feedback

---

## Remote Controls

| Button | Function |
|---------|----------|
| ON | Power ON/OFF |
| MENU | Open Main Menu |
| UP | Navigate Up |
| DOWN | Navigate Down |
| OK | Select |
| LEFT | Previous Student |
| RIGHT | Next Student |
| BACK | Return |
| RESET | Reset Attendance |
| MUTE | Toggle Buzzer |
